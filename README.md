## Credentials
Create a file in the folder _utils_ named credentials.js with this code:
```js
export const credentials = {
    id: 'YOUR_CLIENT_ID',
    sec: 'YOUR_SECRET_ID'
}
```
You can get your credentials from https://github.com/settings/developers --> OAuth Apps.
